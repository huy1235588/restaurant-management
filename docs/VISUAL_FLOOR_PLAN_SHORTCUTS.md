# Visual Floor Plan Editor - Keyboard Shortcuts

## Tool Selection
- **V** - Select Tool (default)
- **H** - Pan Tool
- **T** - Add Table Tool
- **Delete** - Delete selected table

## Actions
- **Ctrl+S** - Save floor plan
- **Ctrl+Z** - Undo
- **Ctrl+Shift+Z** - Redo
- **G** - Toggle grid
- **Esc** - Cancel current tool (Add/Delete mode)

## Tool Usage

### Select Tool (V)
- Click tables to select
- Drag to move
- Resize handles appear on selected tables

### Pan Tool (H)
- Click and drag to pan the canvas
- Useful for navigating large floor plans

### Add Table Tool (T)
- Blue ghost preview follows cursor
- Red when collision detected
- Click to place and open quick create dialog
- Grid snapping when enabled
- Press Esc to cancel

### Delete Tool (Delete key)
- Select a table first
- Press Delete key to open confirmation
- Cannot delete occupied tables
- Warning for reserved tables
- Press Esc to cancel

## View Controls
- **Fit to View** button - Centers all tables
- **Reset View** button - Returns to origin (0,0)
- **Zoom** controls - Zoom in/out or reset to 100%

## Tips
- Hold Shift while dragging to disable grid snapping
- Use Ctrl+S frequently to save your changes
- Ghost preview shows if placement is valid (blue) or has collision (red)
