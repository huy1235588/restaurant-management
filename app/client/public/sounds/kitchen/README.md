# Kitchen Display System - Sound Files

This directory contains audio notification files for the Kitchen Display System.

## Required Sound Files

Place the following MP3 files in this directory:

1. **new-order.mp3** - Plays when a new order arrives
   - Recommended: Short, attention-grabbing sound (1-2 seconds)
   - Suggested: Bell chime or notification tone

2. **order-ready.mp3** - Plays when an order is marked ready
   - Recommended: Pleasant completion sound (1-2 seconds)
   - Suggested: Success chime or ding

3. **urgent.mp3** - Plays for urgent priority orders
   - Recommended: More urgent, attention-grabbing sound (2-3 seconds)
   - Suggested: Alert sound or urgent beep

## Sound Sources

You can find free notification sounds from:
- [Pixabay Sounds](https://pixabay.com/sound-effects/)
- [Freesound](https://freesound.org/)
- [Notification Sounds](https://notificationsounds.com/)

## File Format

- Format: MP3
- Sample Rate: 44.1 kHz
- Bitrate: 128 kbps or higher
- Duration: 1-3 seconds

## Testing

To test the sounds, enable audio in the Kitchen Display settings and trigger events in the UI.

## Volume Control

Users can control the volume through the Kitchen Display interface settings panel.
