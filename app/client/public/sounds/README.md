# Notification Sound

Place a notification sound file named `notification.mp3` in this directory.

The sound will be played when:
- New kitchen orders arrive (Kitchen Display System)
- Important system notifications

Recommended sound characteristics:
- Format: MP3
- Duration: 1-2 seconds
- Volume: Medium (will be played at 50% volume)
- Type: Pleasant chime or bell sound

You can download free notification sounds from:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/notification/
